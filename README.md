## 🚀 Featured projects

### 🔍 DocInsight — Multimodal RAG system
`Python` `LangChain` `ChromaDB` `Tesseract OCR` · *May 2026*

A multimodal RAG pipeline supporting both native and scanned PDFs with page-level source citations. Uses PyMuPDF + Tesseract for ingestion, sentence-transformers for embeddings, ChromaDB for cross-session vector retrieval, and OpenRouter's LLM API for context-grounded answers — fully model-agnostic.

---

### 🎬 Collaborative filtering recommendation system
`Python` `NumPy` `Pandas` `Scikit-learn` · *April 2025*

Built User-KNN, Item-KNN, and Matrix Factorization (SGD) from scratch on MovieLens 100K — no recommender libraries. Matrix Factorization achieved **NDCG@10: 0.093**; Item-KNN hit **RMSE: 0.929** with **71% catalog coverage** vs ~19% for other models.

---

`C/C++` · *Sept 2023*

Discrete-event OS process scheduler with dynamic process generation, Gantt chart visualization, and scheduling algorithms (RR, MLFQ) with customizable time slices.
